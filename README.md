PARA VISUALIZAR EL CONVERTOR DE UNIDADES:
1. click en el boton verde "code" y descargar el archivo en .zip
2. extraer el archivo y meterse a nuevo_proyecto-main
3. meterse en la carpeta convertor, y click a MAIN2
4. en el navegador se abrira el HTML y te pedira opcion 1 o 2. OPCION 1 (CELSIUS Y FARENHEIDT) y OPCION 2(METROS Y KILOMETROS)
5. colocar el numero que se busca cambiar
6. para ver los cambios, presionar F12 o control+shift+i (WINDOWS) ⌘ + Option + J (MAC) y se abrira la consola, donde se mostraran los resultados.
